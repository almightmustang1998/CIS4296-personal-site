Projects

Home Builder Site

This project is a web application that conforms to the multiple-tier software architecture, which works with dynamic data displays and a database to allow a customer to build their dream home. There are user-friendly features such as input validation and data binding, facilitating seamless interaction. The project relies on object oriented programming for modular and scalable design. Enabled server-side processing for updating sales information and calculating costs. Designed a professional UI using CSS tools like Bootstrap, ensuring a polished appearance.

Network Spell Checker

The Network Spell Checker uses multithreading programming techniques and network sockets to communicate between a server thread and client threads to see if a word exists in its dictionary while logging all interactions. This project was built using low-level C.

Car Rental Application

Using the MVC Design Pattern, C#, and Microsoft Access this CRUD application allows employees of a Car Rental business to perform their daily functions such as adding cars to their inventory, modifying details about cars, deleting cars from their inventory, etc. The application also allows customers to view, search, and rent cars through a Windows Form application.
